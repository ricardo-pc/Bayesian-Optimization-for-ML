## Key Design Decisions

### Encoding (why SE kernel works across scales)
- `lr`, `l2`: log-encoded тЖТ `log_lr = log(lr)`, `log_l2 = log(l2)` so distances are meaningful across orders of magnitude
- `emb_dim`: ordinal 0тАУ3 тЖТ {32, 64, 128, 256}
- `mlp_layers`: ordinal 0тАУ2 тЖТ {[64,32], [128,64,32], [256,128,64,32]}
- `alpha`: continuous [0, 1], passed through as-is

### Subprocess Streaming Fix
- Original: `subprocess.run(capture_output=True)` тАФ silently buffered all output until completion
- Fixed: `subprocess.Popen` + line-by-line stdout loop + daemon stderr thread
- This gives epoch-by-epoch NDCG progress visible in the log

### Notebook Decoupling
- Original notebook required live `bo` object in memory
- Fixed: all cells load from CSV, rebuild GP from saved encoded columns
- Notebook is now runnable on any machine without GPU after job completes
